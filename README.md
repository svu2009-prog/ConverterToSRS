# List Processor for sing-box

Десктопное GUI-приложение для обработки списков доменов и IP-адресов с генерацией файлов правил для **sing-box**.

## Возможности

| Функция | Описание |
|---------|----------|
| **Входные форматы** | `.txt`, `.lst`, `.doc`, `.docx` |
| **Кодировки** | UTF-8, UTF-8 BOM, ANSI, Windows-1251 (автоопределение) |
| **Типы списков** | Домены, IPv4, IPv6, смешанный (автоопределение) |
| **Очистка** | Удаление пустых строк, пробелов, табуляции, комментариев (`#`, `;`, `//`) |
| **Валидация** | Проверка доменов, IPv4, IPv6, CIDR |
| **Дубликаты** | Автоматическое удаление с подсчётом |
| **Сортировка** | Домены по алфавиту, IP по возрастанию |
| **Выходные файлы** | JSON (Rule Set v1–v4), LST, SRS (скомпилированный) |
| **sing-box** | Проверка наличия, компиляция SRS, инструкции по установке |
| **Интерфейс** | PySide6, тёмная/светлая тема, Drag & Drop, горячие клавиши |
| **Настройки** | Сохранение последней папки, версии Rule Set, темы, геометрии окна |

## Установка

### Зависимости

```bash
pip install -r requirements.txt
```

### Быстрый запуск (Windows)

Запустите `run.bat` двойным щелчком — скрипт сам установит зависимости и откроет приложение.

### sing-box (для создания SRS)

```bash
winget install sing-box    # Winget
scoop install sing-box     # Scoop
choco install sing-box     # Chocolatey
```

Или скачайте вручную: https://github.com/SagerNet/sing-box/releases

## Запуск

```bash
python main.py
```

## Использование

1. **Откройте файл** — нажмите **Обзор** (`Ctrl+O`) или перетащите файл в окно
2. **Автоопределение** — программа определит тип списка и покажет статистику
3. **Невалидные записи** — при наличии ошибок появится диалог для их удаления
4. **Смешанный список** — будет предложено разделить на домены и IP
5. **Настройте параметры** — версия Rule Set, какие файлы создавать
6. **Начните обработку** — нажмите **Начать обработку** (`F5`)
7. **Результат** — файлы сохранятся в ту же папку, откуда открыт исходный файл

### Пример

Файл `domains.txt`:
```
google.com
youtube.com
 google.com  (дубликат)
# комментарий
```

Создаёт:
- `domains.json` — Rule Set для sing-box
- `domains.lst` — текстовый список
- `domains.srs` — скомпилированный Rule Set (при наличии sing-box)

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Ctrl+O` | Открыть файл |
| `Ctrl+S` | Сохранить отчёт |
| `Ctrl+Q` | Выход |
| `F5` | Начать обработку |

## Обработка ошибок

Приложение обрабатывает следующие ситуации:
- Отсутствие sing-box
- Нет доступа к папке
- Повреждённый DOCX-файл
- Неверная кодировка
- Невозможность записи файла
- Ошибка компиляции SRS
- Файл занят другой программой
