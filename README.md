# List Processor for sing-box

Десктопное GUI-приложение для обработки списков доменов и IP-адресов с генерацией файлов правил для **sing-box**.

## Возможности

- **Чтение файлов**: `.txt`, `.lst`, `.doc`, `.docx` (UTF-8, UTF-8 BOM, ANSI, Windows-1251)
- **Автоопределение типа списка**: домены, IPv4, IPv6, смешанный
- **Очистка**: удаление пустых строк, пробелов, комментариев (`#`, `;`, `//`)
- **Валидация**: проверка корректности доменов и IP-адресов (включая CIDR)
- **Удаление дубликатов** и **сортировка** (домены по алфавиту, IP по возрастанию)
- **Генерация файлов**:
  - JSON (Rule Set, версии v1–v4)
  - LST (текстовый список)
  - SRS (скомпилированный Rule Set через `sing-box rule-set compile`)
- **Проверка sing-box**: поиск в PATH или рядом с программой
- **Drag & Drop**: перетаскивание файлов в окно
- **Горячие клавиши**: `Ctrl+O`, `Ctrl+S`, `Ctrl+Q`, `F5`
- **Темы**: тёмная (по умолчанию) и светлая
- **Сохранение настроек**: последняя папка, версия Rule Set, тема
- **Журнал**: встроенный лог с возможностью сохранения в TXT
- **Отчёт**: детальный отчёт об обработке с экспортом в TXT/JSON

## Установка

### Зависимости

- Python 3.12+
- PySide6
- python-docx (для `.doc`/`.docx`)
- chardet (для определения кодировки)

```bash
pip install -r requirements.txt
```

### sing-box (для создания SRS)

Установите [sing-box](https://github.com/SagerNet/sing-box/releases) одним из способов:

```bash
winget install sing-box    # Winget
scoop install sing-box     # Scoop
choco install sing-box     # Chocolatey
```

Или скачайте архив вручную, распакуйте и добавьте папку в `PATH`.

## Запуск

```bash
python main.py
```

## Использование

1. Нажмите **Обзор** или перетащите файл в окно
2. Программа автоматически определит тип списка
3. Настройте параметры (версия Rule Set, какие файлы создавать)
4. Нажмите **Начать обработку** (`F5`)
5. Готовые файлы сохранятся в ту же папку, откуда открыт исходный файл
